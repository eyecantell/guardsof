# Guardian Training: Preparing for Empathetic Outreach

## Purpose
To equip Guardians with essential knowledge, skills, and mindset for safe, compassionate outreach. Training focuses on ethics, validation, crisis handling, workflow, and long-game resilience-keeping it concise yet effective. All guardians complete core training before outreach.

## Training Structure
- **Format**: Self-paced videos + one live interactive session (Zoom/in-person).
- **Duration**: 
  - Core videos: ~45-60 minutes.
  - Live role-play/Q&A: ~45-60 minutes.
  - Total initial: 1.5-2 hours (spread over 1-2 sessions if needed).
- **Delivery**: Video modules (recorded for replay) + live facilitation.
- **Who Trains**: Project founder/moderators initially; senior guardians/partners as we scale.
- **Certification**: Complete videos + live session participation; ongoing small group involvement required.
- **Ongoing**: Quarterly 30-min refreshers + small group discussions.

## Core Content Outline
### Self-Paced Videos (~45-60 min total)
1. **Mission and Mindset (10-15 min)**:
   - Overview: Proactive hashtag-based outreach for hope and church fellowship.
   - Long-game: "Success often takes months-one church connection changes eternity. Celebrate small wins; rejection is normal sowing."
   - Biblical encouragement: Parable of the sower; loving the brokenhearted (Psalm 34:18).

2. **Ethics and Boundaries (10 min)**:
   - Key guidelines: Consent, privacy, non-intrusive (review guardian_faq.md highlights).
   - OAuth simple: Optional for auto-logging-reads only your public tweets.
   - When to disengage: Minors, advice-giving, pressure.

3. **Practical Workflow (15 min)**:
   - guardsof.com demo: Hashtag queue, opener suggestions, copy/paste + deep links, CRM basics, follow-ups.
   - Message tips: Validation-first, brief/warm; good/bad examples.

4. **Crisis Basics (10 min)**:
   - Indicators + immediate response: Validate, redirect to pros (988).
   - Full details in separate crisis_protocol.md (quick-reference link in dashboard).

### Live Interactive Session (~45-60 min)
- **Empathetic Communication**: Situation-based styles + validation practice.
- **Role-Playing**: 3-5 scenarios (positive chat, rejection, crisis hint, resource share).
- **Q&A/Debrief**: Address concerns, reinforce long-game support.

## Additional Resources
- Dashboard quick-references: Sample messages, crisis_protocol.md link, FAQ responses.
- Small groups: Weekly/monthly for ongoing practice, prayer, debrief.

This training is designed to be accessible and efficient while building confidence and resilience for the long haul.
